# EduAIHub Frontend Redesign Brief

Project: EduAIHub

Goal:
Redesign the frontend into a distinctive, slightly unconventional AI-learning platform for Vietnamese students. Avoid generic SaaS/AI dashboard aesthetics.

Core pages:
1. Home
2. AI Tools
3. Lessons
4. Resources
5. Prompt Guide
6. About / Project Mission

Main idea:
EduAIHub helps students learn how to use AI effectively. The platform should teach:
- how to use AI tools
- how to write better prompts
- how to apply AI to learning, research, writing, coding, English, and productivity

Design direction:
Bold, educational, futuristic but not corporate. Use layered cards, asymmetric layout, strong typography, subtle motion, and clear learning pathways.

Do not:
- Do not change backend logic
- Do not change database schema
- Do not break routing
- Do not delete existing features without asking
- Do not create a generic blue-purple SaaS landing page

Technical requirements:
- Keep the current framework and component structure if possible
- Make responsive UI for mobile/tablet/desktop
- Improve accessibility
- Use reusable components
- Keep code clean and easy to maintain
- Run lint/build after changes
